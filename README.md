The Release contains two files, MSC.zip and MSC.py

If you have Python installed and know how to run .py files, download the MSC.py
If you don't know, download the MSC.zip



Python: 
Download: https://www.python.org/ftp/python/3.14.4/python-3.14.4-amd64.exe (Not compatible? > https://www.python.org/downloads/)

You can use softwares (i.e VisualStudio) OR command promt to run MSC.py

How to run using Command Promt: Locate the folder it is located in (usually Downloads), type, cd Downloads and Then type, python3 MSC.py
If its not located in Downloads, on Command Promt type, cd <filelocation> and type, python3 MSC.py
